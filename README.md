# The Local Guide: Chennai Street Food

## Problem
Tourists and locals struggle to decide what street food to eat based on time, taste, and location.

## Solution
A simple local guide powered by Kiro that recommends Chennai street food using a custom context file (`product.md`).

## How Kiro Helped
Kiro used the local knowledge provided in `product.md` to understand:
- Popular foods
- Cultural preferences
- Best locations

This reduced development time and improved accuracy.

## Tech Stack
- HTML
- JavaScript
- Kiro (context-based prompting)
